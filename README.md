# exhentai crawler

exhentai 爬虫。

排行数据来自表站的 [EHG Toplists](https://e-hentai.org/toplist.php)。

## 使用

复制 `config.TEMPLETE.ini` 为 `config.ini`，自行添加 cookies，修改配置文件。

使用命令行方式运行：

```shell script
python base.py
```
